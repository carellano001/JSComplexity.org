About this site

JSComplexity.org runs on Node.js and Express, using Hogan-Express for template rendering and Stylus to generate CSS. The source code is available on GitHub, under the MIT license. The title font is Ostrich Sans by Tyler Finck, downloaded from The League of Moveable Type. The background images are Subtle Dark Vertical by Cody L and Gray Lines by Atle Mo, both downloaded from Subtle Patterns.

The complexity reporting is performed by complexityReport.js, a Node.js-based tool that analyses syntax trees generated by the Esprima JavaScript parser. If you want to analyse a large codebase, you might find it easier to install complexityReport.js on a development machine or CI server and run the tool from the command line. Detailed instructions for doing so can be found in the readme file.
