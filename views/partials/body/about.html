About this site

JSComplexity.org runs on Node.js and Express, using Hogan-Express for template rendering and Stylus to generate CSS. The source code is available on GitHub, under the MIT license. The styling was done by genius front-end web developer extraordinaire, Perry Harlock.

The complexity reporting is performed by complexityReport.js, a Node.js-based tool that analyses syntax trees generated by the Esprima JavaScript parser, available from npm under the package name complexity-report. If you want to analyse a large codebase, you might find it easier to install complexityReport.js on a development machine or CI server and run the tool from the command line. Detailed instructions for doing so can be found in the readme file.
